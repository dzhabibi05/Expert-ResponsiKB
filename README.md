# Sistem Pakar Penyakit Tanaman Cabai

Sistem ini adalah sebuah aplikasi berbasis web (Sistem Pakar) yang digunakan untuk mendiagnosis penyakit pada tanaman cabai berdasarkan gejala-gejala yang terlihat. Aplikasi ini dibangun dengan tujuan membantu petani atau pengguna dalam mengidentifikasi masalah pada tanaman cabai mereka secara dini serta mendapatkan saran penanganan yang tepat.

## Fitur Utama
- **Diagnosis Penyakit**: Pengguna dapat memilih gejala-gejala yang dialami oleh tanaman cabai, dan sistem akan menyimpulkan penyakit yang paling mungkin.
- **Saran Penanganan**: Memberikan rekomendasi solusi dan penanganan terkait penyakit yang terdiagnosis.

## Teknologi yang Digunakan
- **Backend**: Python (Flask)
- **Frontend**: HTML, CSS, JavaScript

## Cara Menjalankan Aplikasi

1. Pastikan Python sudah terinstal di komputer Anda.
2. Buka terminal/command prompt dan arahkan ke direktori proyek ini.
3. Install semua dependensi yang dibutuhkan dengan perintah:
   `
   pip install -r requirements.txt
   `
4. Jalankan aplikasi menggunakan perintah:
   `
   python app.py
   `
5. Buka web browser Anda dan akses aplikasi di http://127.0.0.1:3000 (atau sesuai dengan port localhost yang tertera di terminal).
